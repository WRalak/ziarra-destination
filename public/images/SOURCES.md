# Demo image sources

Development imagery is downloaded from country-specific [Unsplash](https://unsplash.com) search result pages by `scripts/download-demo-images.ps1` and processed by Unsplash's image CDN into fixed-size WebP files.

Unsplash permits these photos to be downloaded and used under the [Unsplash License](https://unsplash.com/license). The script uses stable photo CDN URLs—not random or changing image endpoints—and never sources imagery from Ziarra production or another travel operator.

Search collections: Kenya, Uganda, Tanzania, Rwanda, South Africa, Namibia, Botswana, Zambia, Ethiopia, Morocco, Egypt, and Seychelles travel photography.
